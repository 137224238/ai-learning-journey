# Agent Survey 精读

> Wang et al., "A Survey on Large Language Model based Autonomous Agents"
> arXiv: 2308.11432

## Agent 四大组成
1. **Profile**：角色定义
2. **Memory**：记忆（短期/长期/外部）
3. **Planning**：规划（CoT/ReAct/ToT）
4. **Action**：工具调用 / 代码执行

## 关键洞察
- Planning 是当前最大瓶颈
- 多 Agent 协作 > 单 Agent（复杂任务）
