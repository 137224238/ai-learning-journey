### 第 1-3 个月：构建核心能力

**重点**：打牢基础，避免走捷径

```
月度目标分解：

第 1 个月：01-foundations + 02-llm-core（前半）
├── Week 1-2: Python 进阶 + ML 基础
├── Week 3-4: 深度学习 + Transformer 手写
└── 验证：能独立解释 Attention 机制并写出代码

第 2 个月：02-llm-core（后半）+ 03-rag-systems
├── Week 5-6: Prompt 工程 + 评估指标
├── Week 7-8: Naive RAG → Advanced RAG
└── 验证：搭建一个本地可运行的 RAG 系统

├── Week 9-10: ReAct Agent + Tool Use
├── Week 11-12: 完成企业文档问答系统
└── 验证：项目有 README、可演示、有评估结果
```

### 第 4-6 个月：项目打磨 + 求职准备

```
第 4 个月：完成第 2、3 个项目
├── 代码审查 Agent
└── 个人学习助理

第 5 个月：07-interview-prep（重点月）
├── 系统设计题（每天 1 题）
├── ML 面试题（每周 10 题）

第 6 个月：全力求职
├── 简历打磨（投递前 2 周）
├── 开始投递（目标：每周 20 份）
```

### 第 7-12 个月：入职提升

- 入职后：持续学习论文、参与开源项目
- 目标：从初级 → 中级 AI 工程师
